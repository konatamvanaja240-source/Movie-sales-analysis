# Movie-sales-analysis
## Project Overview

This project presents an interactive Power BI dashboard that analyzes movie sales data across multiple dimensions such as revenue, genre, ratings, worldwide sales, and regional contribution.
The goal of this project is to derive meaningful insights from the dataset and visualize key trends to support business decision-making.

 ## Objectives

- To analyze and visualize movie performance across different regions.
- To identify top-performing genres and highest-grossing movies.
- To study the correlation between ratings and revenue.
- To create an interactive dashboard for easy exploration of insights.
## Tools & Skills I Used
## Tools
- Power BI Desktop – Data modeling, report creation, interactive dashboards
- Power Query – Data cleaning, transformation, and preprocessing
- DAX (Data Analysis Expressions) – Creating measures, calculated columns, and KPIs
- Excel/CSV Sources – Raw data storage and basic preprocessing
- Power BI Visualization Tools – Charts, slicers, filters, drilldowns
## Skills
- Data Cleaning & Transformation
- Data Modeling (Star Schema relationships)
- KPI Development using DAX
- Dashboard Design & Visualization
- Analytical Storytelling
-Trend & Comparative Analysis
- Understanding of Box Office Metrics

##  Key Insights (summary)

- Year-wise trend analysis of movie sales.
- Comparison of revenue performance by genre.
- Global vs regional sales distribution.
- Relationship between IMDb ratings and box-office success.
- Top 10 movies based on worldwide revenue.
## Conclusion
This Power BI project provides meaningful insights into movie sales by analyzing trends, genres, ratings, and regional performance. The interactive dashboard enables quick decision-making and highlights key revenue drivers. Overall, the report enhances understanding of box office dynamics and supports data-driven strategic planning in the entertainment industry.
